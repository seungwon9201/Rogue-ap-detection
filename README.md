데이터셋(Wireshark를 통한 데이터 수집)
SSID: 와이파이 네트워크 이름.
BSSID: 액세스 포인트의 MAC 주소.
채널: 와이파이가 사용하는 주파수 채널.
신호 세기: dBm 단위로 측정된 신호 강도.
암호화 방식: 와이파이에 사용된 암호화 유형(Open, WEP/WPA/WPA2 등).
지원 속도: 와이파이가 지원하는 속도 목록.
비콘 간격: 비콘이 전송되는 주기.
예상 RTT: 예측된 왕복 지연 시간.
벤더 OUI: 벤더의 OUI(Organization Unique Identifier).
